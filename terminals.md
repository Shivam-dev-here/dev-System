## Terminals, Bash, Shell, Linux

- CMD                                                  : Default Windows terminal
- PowerShell                                         : Advanced Windows terminal
- Unix Shell (Bash / Zsh)                       : Linux / Mac terminal
- Git Bash                                             : Bash on Windows
- WSL (Windows Subsystem for Linux): Full Linux inside Windows

---
## Terminal Navigation & File System

### Bash Commands

- `# i am a comment` :  comment in bash
- `pwd` : Print Working directory
- `ls`   : List of present files and folders
- `cd "Virtual Painter"`  : Moves inside a folder from working directory
- `cd ..` : Move backward
- `cd "Virtual Painter/Mini Project UI"` : jumps to a folder
- `code img.jpeg`  : Opens a file in vs code
- `python virtualpaint.py`  : Runs a code
- `clear` : clears bash

### CMD Commands

- `REM i am a comment`   : comments in cmd
- `cd`     : Current directory
- `dir`       :  List files and folders  
- `cd "Virtual Painter"`  : Move Forward 
- `cd ..`   : Move backward
- `cd "Virtual Painter\Mini Project UI"`  :  Jump to folder  
- `code img.jpeg`    :  Open file in VS Code 
- `python virtualpaint.py` :  Run Python file  
- `cls`  :    Clear terminal
